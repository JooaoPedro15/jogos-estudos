# AEDS II Card Roguelike

Jogo educativo em React + TypeScript para treinar questoes de AEDS II no estilo da prova. A run usa cartas de conceitos para resolver encontros baseados em ABB, AVL, arvore alvinegra, hash e TRIE.

## Como rodar

```bash
npm.cmd install
npm.cmd run dev
```

No ambiente atual o servidor esta rodando em:

```text
http://localhost:52861/
```

## Comandos uteis

```bash
npm.cmd run test
npm.cmd run lint
npm.cmd run build
```

## Conteudo do MVP

- 14 encontros em sequencia, incluindo normais, elites e chefe.
- Banco de desafios com arvore binaria generica, ABB, AVL, alvinegra, hash, TRIE e doidona.
- Cartas de caso base, percurso, condicao, combinacao, modificador e complexidade.
- Combos que recompensam padroes de solucao como busca em ABB, travessia completa, pesquisa com overflow e busca em TRIE.
- Combos para navegacao em estruturas hibridas com T1, T2 e lista encadeada.
- Passivas de run com bonus, desconto de energia, multiplicador e protecao contra perda de foco.
- Interface jogavel com foco, energia, moedas, score, recompensas e diagramas SVG das estruturas.
